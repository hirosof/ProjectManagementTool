# Phase 5（Textual版）設計中 論点隔離スレッド【A】

本スレッドは、
プロジェクト管理ツール **Phase 5（Textual版）** の設計作業中に発生する  
**判断が割れそうな論点・設計上の迷い・未整理事項を一時的に隔離するための Aスレッド**です。

本スレッドの内容は **確定情報ではありません**。  
マスタースレッドへの直接転記は禁止します。

---

## 本スレッドの位置づけ（重要）

- Phase 5（Textual版）の **設計中専用**
- 思考途中・仮説・違和感・迷いを書いてよい
- 結論を出すことは目的としない
- マスターを守るための **隔離ゾーン**

---

## 本スレッドで扱うもの（Do）

- Textual UI 設計中に発生した判断分岐点
- CLI版に委譲すべきか、UIで扱うべきか迷う事項
- dry-run / 確認UIの粒度に関する悩み
- 設計書には書いていないが、決めないと進めない点
- 「この判断を独断で決めるのは怖い」と感じた事項

---

## 本スレッドで扱わないもの（Don't）

- 最終結論・確定方針
- 設計書の完成文面
- 実装コードレビュー
- フェーズ分解のやり直し

※ 結論が必要になった場合は、  
　**Bスレッド（設計整理）へ昇格**する。

---

## 記載ルール（自由だが推奨）

以下の形式は **推奨**であり、強制ではない。

```

### 論点タイトル（仮）

* 何に迷っているか：
* 現在の案：

  * 案A：
  * 案B：
* 判断が割れそうな理由：
* Phase 5 の思想との関係：

  * 「安全に増やすUI」と矛盾するか？
* 今すぐ決める必要があるか：

  * ある / ない

```

未整理なメモ書きでも問題ない。

---

## A → B → マスター の原則（再掲）

- **A**：迷い・思考・未整理（このスレッド）
- **B**：整理・判断・方針確定
- **マスター**：確定事項のみ記録

途中段階を飛ばさない。

- マスターに直接書かない
- 「迷ったらまずここに書く」でOK
